## 3.1 — SMS vs EMAIL: KHI NÀO DÙNG GÌ?

| Yếu tố | Email 📧 | SMS 📱 |
|---------|----------|--------|
| **Mục đích chính** | Storytelling, relationship building | Real-time updates, urgency, action triggers |
| **Tone** | Structured storytelling, emotional arcs | Conversational, chaotic, live commentary |
| **Length** | 500-800 words | 100-300 words (SMS limit: ~160 chars/segment) |
| **Frequency** | 1/ngày (daily broadcast) | 1-3/tuần (non-sale), 3-8/ngày (sale day) |
| **Structure** | Hook → Story → Bridge → CTA → P.S. | Punch → Update → CTA → Signoff |
| **CTA** | 2-3 CTAs throughout | 1 link, direct |
| **Best for** | Story, education, warmup | Urgency, drops, real-time chaos, reminders |
| **Personality** | Julie the storyteller | Julie the live commentator |

### Golden Rule:
> **Email builds the WANT. SMS triggers the ACTION.**

## 3.2 — AUDIENCE SEGMENTATION: 3-TIER SYSTEM

### Overview

```
┌──────────────────────────────────────────────────────┐
│                   FULL EMAIL LIST                     │
│                  (~55,000 người)                      │
│                                                       │
│    ┌──────────────────────────────────────────┐       │
│    │          SMS MASS SUBSCRIBERS             │       │
│    │           (~10,000-15,000)                │       │
│    │                                           │       │
│    │    ┌─────────────────────────────┐        │       │
│    │    │    HIGH ENGAGE SEGMENT      │        │       │
│    │    │     (~3,000-5,000)          │        │       │
│    │    │                              │       │       │
│    │    │   ┌──────────────────┐       │       │       │
│    │    │   │   REGISTERED     │       │       │       │
│    │    │   │  (~1,000-2,000)  │       │       │       │
│    │    │   └──────────────────┘       │       │       │
│    │    └─────────────────────────────┘        │       │
│    └──────────────────────────────────────────┘       │
└──────────────────────────────────────────────────────┘
```

### Chi tiết từng tier

| Tier | Ai? | Họ nhận gì? | Tone | Volume |
|------|-----|-------------|------|--------|
| **Mass** | Tất cả SMS subscribers | Teaser ngắn, urgency, "join the action" | Brief, punchy, CTA-focused | Thấp: 3 SMS/chiến dịch |
| **High Engage** | Opens > 50%, clicks nhiều, mua > 2x | Story-driven updates, behind-the-scenes, exclusive previews | More personal, narrative-style | Trung bình: 5 SMS/chiến dịch |
| **Registered** | Đã đăng ký tham gia sale cụ thể (opt-in keyword) | Real-time sale updates, live order counts, sold-out alerts | Live commentator, chaos narration, celebration | Cao: 3 warmup + 6-8 sale day |

### Tại sao phải segment?

1. **Mass nhận ít** → tránh unsubscribe từ over-messaging
2. **High Engage tưởng thưởng** → exclusive content = loyalty
3. **Registered = VIP** → họ đã opt-in vào EVENT → muốn nhận nhiều updates

## 3.3 — TEXT-TO-JOIN SYSTEM

### Cách thu thập SMS subscribers theo event

| Element | Ví dụ |
|---------|-------|
| **Keyword** | SKULL995, SKULL995F, F13 |
| **Format** | Text [KEYWORD] to [number] |
| **Where to promote** | Email P.S., daily broadcasts, landing pages, social |
| **Confirmation SMS** | Ngay lập tức khi opt-in |

### Confirmation Message Template

```
Welcome to the [EVENT] VIP list, witch. 🖤

Here's what's coming:
- [Date]: Sale drops at [time]
- [Pricing teaser]: Starting from $X.XX
- This number gets the link FIRST.

Save this number. When I text you the link — move FAST.

— Julie
(you just joined [X] other witches. it's getting crowded in here.)
```

### Keyword Strategy cho Promotions

| Campaign | Keyword | Khi nào push |
|----------|---------|-------------|
| SKULL-A-PALOOZA | SKULL995 | Teaser emails E1-E5 |
| During sale (new drop alerts) | SKULL995F | Sale Day email P.S. |
| Friday the 13th | F13 | Teaser phase |
| Generic VIP list | SKULL | Always in email footer |

## 3.4 — 6 SMS TYPES & TEMPLATES

### Type 1: TEASER SMS

**Khi nào:** Pre-sale, 1-2 tuần trước event
**Mục đích:** Tạo curiosity, push keyword opt-in
**Tone:** Quick, mysterious

```
Template:
[1 câu hook gây tò mò]
[1 câu teaser offer]
[CTA: Text KEYWORD to NUMBER]

Ví dụ (SM1 — SKULL-A-PALOOZA):
Something big is dropping Mar 17.
Skull bags from $9.95. I'm not kidding.
Text SKULL995 to 1-866-580-9334 to get the link FIRST.
— Julie 💀
```

**Khi nào:** Sale day, real-time
**Mục đích:** FOMO, urgency, live narration

```

[Specific numbers: orders, items gone, time elapsed]
```

## 3.5 — SMS CADENCE BY CAMPAIGN PHASE

### Pre-Sale (Teaser Phase)

| Day | Mass | High Engage | Registered |
|-----|------|-------------|------------|
| D-7 | 1x Teaser | — | — |
| D-5 | — | 1x Story teaser | — |
| D-3 | — | — | 1x "You're on the list" |
| D-2 | 1x Urgency | 1x Last call | 1x Warehouse preview |
| D-1 | 1x Last call | — | 1x "Tomorrow [time]. Be ready." |

### Sale Day

| Time | Registered |
|------|-----------|
| Sale launch | "IT'S LIVE" + link |
| +15-30 min | Early chaos update |
| +45-60 min | Record/milestone |
| Mid-day | Scoreboard + reshuffle |
| Afternoon | Countdown begins |
| Evening | Urgent countdown |
| Pre-close | Celebration / last call |
| Close | "I LOVE YOU" / final |

### Frequency Rules

| Tier | Daily max | Campaign max | Cool-down between SMS |
|------|----------|-------------|----------------------|
| Mass | 1 | 3 | 48 hours |
| High Engage | 1 | 5 | 24 hours |
| Registered (warmup) | 1 | 3 | 24 hours |
| Registered (sale day) | 8 | 8/day | 1-2 hours |

## 3.6 — SMS WRITING RULES

### DO ✅

1. **Lowercase everything** (ngoại trừ CAPSLOCK drama) — feels like texting
2. **Short paragraphs** — 1-2 lines max
3. **Specific numbers** — "SIXTEEN bags", "SIX MINUTES", "284 left"
4. **Character reactions** — bao giờ cũng có Alice/Jamie moment
5. **Parenthetical sign-offs** — "(eating popcorn watching this chaos unfold)"
6. **ONE link** — link phải click-ready, không hide behind text
7. **Emojis có mục đích** — 💀🖤⏰🚨 cho brand feel, không spam

### DON'T ❌

1. **Không copy-paste email content** — SMS là kênh RIÊNG
2. **Không dùng formal language** — "Dear customer" = instant death
3. **Không many links** — 1 link duy nhất, rõ ràng
4. **Không gửi SMS chỉ để "nhắc"** — mỗi SMS phải có NEW information
5. **Không exceed frequency rules** — unsubscribe rate tăng = mất revenue
