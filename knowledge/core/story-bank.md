## PIPELINE LEGEND

| Status | Định nghĩa |
|--------|-----------|
| **SEED** | Raw idea — chỉ có title + nguồn gốc. Chưa phát triển. |

| **READY** | Đầy đủ: angle + hook + pivot + product fit. Viết được ngay. |
| **USED** | Đã publish. Có ngày gửi + kết quả (nếu có). |

## ADDING NEW ITEMS

### Format for SEED:
```
**[Title]**
- Source: [Story Source type]
- Raw idea: [1-2 sentences]
- Added: [date]
```

### Format for CONCEPT:
```
**[Title]**
- Source: [Story Source type]
- Angle: [What's the story? What's the Turn?]
- Pillar: [Email Pillar 1/2/3/4]
- Hook direction: [First line idea]
- Product fit: [Which product/offer connects?]
- Added: [date]
```

### Format for READY:
```
**[Title]**
- Source: [Story Source type]
- Angle: [Full story setup]
- Hook: [Exact or near-exact first line]
- Turn: [The unexpected moment]
- Pivot: [Bridge to product]
- Product: [Specific product + offer if any]
- Pillar: [Email Pillar 1/2/3/4]
- Notes: [Any timing constraints, character to feature]
- Added: [date]
```
