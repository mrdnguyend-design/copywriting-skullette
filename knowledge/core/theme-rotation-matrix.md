## Rotation Rules

1. **No same theme 2 days in a row**
2. **Minimum 4 different themes per week**
3. **Gothic Mundane + Witch Victory = workhorses** (2x each = floor)
4. **Brenda max 1x/week** — overuse dilutes polarization power
5. **Industry Commentary max 1x/week** — use sparingly for impact
6. **Cultural Darkness ≥ 1x/week** — tie to real cultural moments when possible
7. **Julie Personal needs development** — only 3 emails to date, huge untapped potential
