## Step 2: CHARACTER CONSISTENCY CHECK

For each character appearing in the draft:

### JULIE
- [ ] First-person voice throughout?
- [ ] Unapologetic? No validation-seeking?
- [ ] (Parenthetical inner thoughts) present?
- [ ] CAPSLOCK for emphasis (not overuse)?
- [ ] Indifferent to Basic Brenda — not angry, not explaining, just darkly amused?
- [ ] Never apologizes for gothic aesthetic?
- [ ] References feel authentic: candles, skulls, combat boots, dark romance, full moons — NOT forced?
- [ ] "Stay spooky" / "Stay wicked" / "Stay weird" sign-off? (one of these variants)

### ALICE (if featured)
- [ ] Competent, practical, enables Julie's chaos?
- [ ] Never the star — supports Julie's story?
- [ ] "Color-coded spreadsheets" level of organization?
- [ ] Energetic reactions are specific and believable?

### JADE (if featured)
- [ ] Design philosophy language? ("every buckle has meaning", "wine-dark linings")
- [ ] Focus is on product quality and aesthetics?
- [ ] Never used as a prop — has actual design perspective?

### BASIC BRENDA (if featured)
- [ ] Archetype mockery, not specific real person?
- [ ] Treated with INDIFFERENCE, not cruelty or hate?
- [ ] Used for polarization purpose only?
- [ ] Would a Basic Brenda reading this feel targeted cruelly vs. just... not addressed?
- [ ] Brenda count this week: ≤ 2x? (flag if approaching limit)

### THE WITCH (customer, if featured)
- [ ] Raw customer words preserved (no paraphrasing)?
- [ ] Her story celebrated, not condescended to?
- [ ] Her darkness treated as normal, not exotic?

## ZERO TOLERANCE BLOCKS

These automatically block publishing until reviewed:

1. **Real people mocked specifically** — Basic Brenda is archetype only. Naming/describing real people = BLOCK.
2. **Cultural facts wrong** — Gothic/dark culture references must be accurate. Wrong book title, wrong cultural reference = BLOCK.
3. **Character violates their core identity** — Julie apologizing, Brenda portrayed sympathetically without purpose = BLOCK.
4. **Audience condescension** — Content that makes the reader feel judged for being a witch = BLOCK.
